This is a simple Coffee Shop website built using HTML, CSS, and basic JavaScript. It includes multiple pages such as Home, Login, and Sign Up, along with a responsive card-based layout to display coffee images.

The project focuses on learning frontend development basics like layout design, navigation, forms, and simple validation.

🚀 Features
🏠 Home page with welcome message
🍔 Navigation bar with links (Home, Login, Sign Up, Contact, About)
📋 Login and Sign Up forms
📱 Responsive card layout using Flexbox
🖼️ Coffee image cards
✔️ Basic form validation using JavaScript
🎨 Styled using external CSS
🛠️ Technologies Used
HTML5
CSS3
JavaScript (basic)
Flexbox for layout
